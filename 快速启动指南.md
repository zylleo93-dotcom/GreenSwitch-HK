## 快速启动

### 方法 1: 双击 START.bat
直接双击项目根目录下的 `START.bat` 文件

### 方法 2: 手动启动

```powershell
# 1. 进入项目目录
cd C:\Users\94799\Desktop\GreenSwitch-HK-Enhanced

# 2. 安装前端依赖
cd frontend
npm install

# 3. 安装后端依赖
cd ..\backend
npm install

# 4. 启动后端 (新终端)
cd backend
npm start

# 5. 启动前端 (新终端)
cd frontend
npm run dev
```

## 访问地址
- **前端**: http://localhost:5173
- **后端 API**: http://localhost:3001

## Firebase Web App 配置 (可选)

如果需要完整的 Firebase 认证功能：

1. 打开 https://console.firebase.google.com/project/sme01-b5910/settings/general
2. 点击 "</>" 图标添加网页应用
3. 输入应用昵称 (如 "GreenSwitch Web")
4. 点击注册应用
5. 复制 firebaseConfig 中的值到 `frontend/.env.local`
